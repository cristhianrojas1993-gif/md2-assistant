# MD2 Assistant — Alpha 0.1

Asistente offline para **Massive Darkness 2**, actualmente con:

- Pícaro
- Explorador
- Árboles de habilidades marcables
- Narrador por voz
- Asistente local de reglas
- Gestión de fichas del Pícaro
- Gestión del mazo de flechas del Explorador
- Historial y guardado local
