# MD2 Assistant Alpha 0.5

Incluye:
- Árbol corregido del Pícaro y Explorador.
- Sombras y Veneno como pasivas.
- Una elección de habilidad por nivel.
- Bloqueo por nivel.
- Reemplazo por grado superior dentro de una rama.
- Costes de XP y subida automática de vida/maná.
- Recordatorios de tesoro.
- Movimiento con PM.
- Ataque guiado en 4 pasos.
- Tiro rápido, certero o fallido para el Explorador.
